# fantasy-rd2l
Reddit Dota 2 Fantasy Points Bot for Discord



# How to install and test

1) Get discord bot token
2) Get Discord Bot added to test server
3) node .createdatabase
4) node .import_csv.js
5) node . 

Your bot should be online
